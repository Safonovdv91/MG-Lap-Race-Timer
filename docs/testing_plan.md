# План тестирования ИК сенсоров с модуляцией

## Цели тестирования
- Проверка корректности детектирования пересечения ИК луча
- Оценка точности измерения времени
- Проверка устойчивости к внешним помехам
- Тестирование дальности и надежности соединения

## Этапы тестирования

### 1. Функциональное тестирование
- Подключение ИК передатчиков и приемников согласно схеме
- Проверка включения/выключения передатчиков
- Проверка реакции приемников на пересечение луча
- Проверка корректности обработки прерываний
- Проверка работы всех режимов измерения (SPEEDOMETER, LAP_TIMER, RACE_TIMER)

### 2. Тестирование точности времени
- Сравнение времени срабатывания с механическими контактами
- Измерение задержки между фактическим пересечением и регистрацией события
- Проверка стабильности временных характеристик

### 3. Тестирование устойчивости
- Проверка работы при ярком окружающем свете
- Проверка на помехи от других ИК источников (телевизоры, другие пульты)
- Проверка работы при различных температурах
- Тестирование на электромагнитные помехи

### 4. Тестирование дальности
- Проверка работы на различных расстояниях (1м, 3м, 5м, 10м)
- Проверка влияния углового отклонения на надежность соединения
- Тестирование работы при частичной засветке луча

### 5. Долговременное тестирование
- Непрерывная работа в течение нескольких часов
- Проверка стабильности работы передатчиков
- Мониторинг температуры компонентов
- Проверка энергопотребления

## Диагностические функции
- Веб-интерфейс для мониторинга состояния ИК системы
- Логирование всех событий пересечения луча
- Отображение уровня сигнала (если возможно)
- Функция самодиагностики при запуске

## Критерии успешного тестирования
- 100% детектирование пересечения луча при правильной установке
- Временная точность не хуже 1 мс
- Устойчивая работа при нормальных условиях освещения
- Дальность действия не менее 3 метров
- Отсутствие ложных срабатываний от бытовых источников света
